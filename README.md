# get-answer

该项目是本人开始学习 vuex 所写的例子。是根据 https://github.com/bailicangdu/vue2-happyfri 这个例子的 UI 来实现的。  
在作者原来基础上新增了多选功能，对于单选功能实现方式也是有所不同的。

## 使用技术

Vue + Vuex + vue-router

## 项目运行

``` bash
# 下载依赖包
npm install

# 开发环境 访问：localhost:8080
npm run dev

# 生产环境
npm run build

# 打包分析
npm run build --report
```
